Stack: Next.js (App Router), Tailwind CSS, Supabase (Postgres + Auth + Realtime + Storage), OpenAI, Vercel.

Quickstart:
1. Create a GitHub repo and push this project.
2. Create a Supabase project and run `supabase/schema.sql`.
3. Add environment variables (see `.env.example`) in Vercel and locally.
4. Install & run locally:
   ```
   npm install
   npm run dev
   ```
5. Deploy to Vercel: import repo, set env vars, deploy.

Security: keep OPENAI_API_KEY and SUPABASE_SERVICE_ROLE_KEY server-side only.
